# CopilotTheory

Esta carpeta contiene material fuente que se utiliza para generar contenido en la web mediante Copilot.

## ⚠️ Importante

- **Esta carpeta está en `.gitignore`** y NO se sube al repositorio
- **La web NO depende de esta carpeta** - todo el contenido generado se integra directamente en el proyecto
- Sirve únicamente como referencia temporal para la generación de contenido

## Uso

1. Sube aquí archivos de teoría, documentación o recursos
2. Copilot los utilizará para generar contenido estructurado en la web
3. Una vez generado el contenido, los archivos pueden permanecer aquí o eliminarse según necesites
